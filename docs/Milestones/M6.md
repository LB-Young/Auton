## 目标

实现可注入领域知识的技能系统。Skill 是一个带 YAML frontmatter 的 Markdown 文件（SKILL.md），**本质是知识文档**，告诉 LLM 在什么场景下该用什么工具、按什么步骤操作。

## 架构设计

### 核心概念：Skill 是知识，不是代码

```
用户输入
    → SkillSearcher 根据 description 做语义检索
    → SkillInjector 选取 top-k 最相关 Skill
    → 将 SKILL.md body 注入 LLM context
    → LLM 根据注入的知识决定调用哪些工具
```

**为什么不是代码**：Skill 的核心是告诉 LLM "何时用、怎么用"，而不是执行特定函数。同一个 Skill 可以跨多个领域复用。

### Frontmatter Schema

```yaml
---
name: github                        # 唯一标识（小写+连字符）
description: "何时使用 / 何时不用    # 最重要字段！LLM 据此判断注入
disable-model-invocation: false    # 禁止 LLM 自动调用
user-invocable: true               # 允许 /skill <name> 触发
load-experiences: true             # 自动加载 experiences/README.md
metadata:
  openclaw:
    emoji: "🐙"
    requires:
      bins: ["gh"]                  # 依赖的二进制命令
    install:
      - kind: brew
        formula: gh
---
```

### 渐进式披露（Progressive Disclosure）

```
元数据（name + description）
    → SKILL.md body（<500 lines，触发后）
    → experiences/（按需，load-experiences=true）
    → references/（按需，LLM 决定）
    → scripts/（直接运行，不入 context）
```

### 技能来源优先级

| 优先级 | 来源 | 路径 |
|--------|------|------|
| 1（最高） | 工作区 | `.auton/skills/`（当前目录） |
| 2 | 项目 | `.auton/skills/`（项目根） |
| 3 | 用户 | `~/.auton/skills/` |
| 4（最低） | 内置 | `src/auton/skills/builtin/` |

同名技能高优先级覆盖低优先级。

### experiences 格式

```markdown
### 2024-01-15: 大表迁移必须用事务包裹
- **场景**：迁移超过 10 万行的表时，直接 ALTER TABLE 导致锁表超时。
- **教训**：大表迁移必须用 BEGIN / COMMIT 包裹分批 UPDATE，或使用 pg_repack。
- **标签**：#migration #performance
```

## 测试方法

### 1. 模块导入验证

```bash
python -c "
from auton.skills import (
    Skill, SkillSource, SkillRegistry, SkillLoader,
    SkillInjector, SkillSearcher, SkillChecker,
    SkillPackager, SkillCreator, parse_skill_file,
)
print('All imports OK!')
"
```

### 2. 技能注册表测试

```bash
python -c "
from auton.skills import SkillRegistry

registry = SkillRegistry.get_instance()
registry.ensure_loaded()
print(f'共 {len(registry)} 个技能')
for s in registry:
    print(f'  {s.source.value}: {s.name}')
"
```

预期：5 个内置技能（skill-creator / github / git-workflow / web-search / code-review）。

### 3. 技能检索测试

```bash
python -c "
from auton.skills import SkillRegistry, SkillSearcher

registry = SkillRegistry.get_instance()
registry.ensure_loaded()
searcher = SkillSearcher(registry)

results = searcher.search('GitHub PR review', top_k=5)
print(f'搜索 GitHub PR review:')
for r in results:
    print(f'  [{r.score:.2f}] {r.skill_name} ({r.match_type})')
"
```

### 4. 技能注入测试

```bash
python -c "
from auton.skills import SkillRegistry, SkillInjector

registry = SkillRegistry.get_instance()
registry.ensure_loaded()
injector = SkillInjector(registry)

content = injector.inject_for_query('review a GitHub PR', top_k=3)
print(content[:400])
"
```

预期：github 和 code-review 技能被注入。
