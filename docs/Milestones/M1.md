## M1 — Core（核心执行闭环）✅ 已完成

**目标**：交付可运行的最小 Agent。

### 已交付功能

| 功能 | 模块 | 文件 |
|------|------|------|
| CLI 入口 | `auton` 命令行工具 | `auton/cli/main.py` |
| 流式/同步执行 | `SessionProcessor.run()` / `run_stream()` | `auton/agent/agent.py` |
| Part 化消息 | TextPart / ReasoningPart / ToolPart / StepPart | `auton/agent/message.py` |
| 事件总线 | EventBus (同步/异步发布订阅) | `auton/core/events.py` |

| MiniMax LLM Provider | MiniMax Anthropic-API 兼容调用 | `auton/llm/minimax_provider.py` |
| 会话持久化 | append-only JSONL 存储 | `auton/agent/session_store.py` |
| 决策策略 | continue / compact / stop | `auton/agent/policies.py` |
| 配置管理 | YAML 配置加载 | `auton/core/config.py` |
| 8 个内置工具 | read / write / edit / glob / grep / bash / web_search / web_fetch | `auton/tools/` |

#### 测试 5：决策策略

```python
from auton.agent.policies import DecisionPolicy, PolicyInput

policy = DecisionPolicy()

# 停止词触发 stop
r = policy.decide(PolicyInput(
    message_count=5, token_count=100,
    last_user_message="再见", step_count=1
))
assert r.status == "stop"

# 正常消息触发 continue
r = policy.decide(PolicyInput(
    message_count=5, token_count=100,
    last_user_message="继续", step_count=1
))
assert r.status == "continue"

print("✓ 决策策略正常")
```

## 快速测试脚本

每次开发新功能后，运行以下命令快速验证 M1 未被破坏：

```bash
#!/bin/bash
# scripts/milestone_test.sh

set -e
source ~/miniconda3/etc/profile.d/conda.sh
conda activate openclaw_env
cd /Users/liubaoyang/Documents/外包/Auton
pip install -e . -q

echo "=== M1 快速测试 ==="

echo "[1] CLI 帮助"
auton --help > /dev/null && echo "✓ CLI OK"

echo "[2] 导入检查"
python -c "from auton.agent import SessionProcessor; from auton.tools import get_default_tools; print(f'{len(get_default_tools())} tools')" && echo "✓ 导入 OK"

echo "[3] MiniMax 流式对话"
export MINIMAX_API_KEY="${MINIMAX_API_KEY:?请设置 MINIMAX_API_KEY}"
python scripts/debug_query.py "你好" --provider minimax --model MiniMax-M2.7 > /dev/null 2>&1 && echo "✓ 流式对话 OK"

echo "[4] JSONL 写入检查"
test -d ~/.auton/memory/execution && echo "✓ 存储目录 OK"

echo "=== M1 快速测试完成 ==="
```

运行：

```bash
export MINIMAX_API_KEY="your-key"
bash scripts/milestone_test.sh
```
